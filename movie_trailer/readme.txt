Creates a static webpage containing my three favorite movies:
-Snatch
-The Big Lebowski
-Wedding Crashers

To use:
1. Place all files in the same folder
2. launch entertainment_center from command line (e.g. python file_path\entertainment_center.py)
3. Enable ActiveX in browser if prompted
4. Click on each movie to launch trailer
5. Enjoy my awesome movie collection :)
